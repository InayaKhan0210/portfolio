about me

A bit of background

My journey into tech started with a simple question: what happens after the code is written? While learning programming and building projects in college, I became curious about how applications are deployed, maintained, and scaled in real-world environments. That curiosity gradually introduced me to DevOps, Linux administration, and cloud infrastructure.

Since then, I've been learning by building. From creating web applications and working with Docker to setting up CI/CD pipelines and exploring cloud technologies, I enjoy gaining hands-on experience through practical projects. One of the things I find most exciting about DevOps is how it brings development and operations together to create reliable and efficient systems.

As a third-year Electronics and Computer Engineering student, I'm continuously expanding my knowledge of modern tools and workflows. Currently, I'm focused on strengthening my skills in containerization, automation, cloud platforms, and deployment practices while working toward a career in DevOps and cloud engineering.
